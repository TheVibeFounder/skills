# TheVibeFounder Skills

Public directory of Claude Code skills built live on TVF Build Live sessions. The deployed site lives at **[thevibefounder.com](https://thevibefounder.com)**.

## What's here

```
/index.html                                          — skills hub
/thevibefounder-social-insta-slide-skill-01-apr-21.html  — first skill detail page
/LICENSE                                             — MIT
```

Every skill follows the naming convention `thevibefounder-<category>-<slug>-skill-<NN>-<date>` so the repo stays self-documenting as more skills ship.

## How people get a skill

1. Visit the detail page
2. Drop their email
3. Get a signed, time-limited download link emailed + auto-triggered

The download bundle itself is hosted on R2 and distributed via a Cloudflare Worker — infrastructure lives elsewhere; this repo is only the public site.
