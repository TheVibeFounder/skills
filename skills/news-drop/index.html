The News Drop — one slide, one claim, one CTA.

A Claude Code skill that generates a single pixel-perfect 1440×1800 Instagram post. Corner cartouche brackets, field-note header, spotlight-bracket headline, stamp footer. Full agent instructions baked in — drop it into any project and ask for a news drop.

How to install
  1. 01Click Download skill on the right (ZIP from GitHub).
  2. 02Unzip into .claude/skills/thevibefounder-social-insta-slide-skill-01-apr-21/ in your project.
  3. 03Install the renderer: npm install + npx playwright install chromium.
  4. 04In Claude Code, ask: "make a news drop about X" — SKILL.md carries the full agent playbook; the skill auto-triggers.
  5. 05Output PNG lands in ./out/ at 1440×1800. Ship.
